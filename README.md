# AnomaliesDetection
This project uses SARIMAX forecasting model to predict the target from timeseries data. It then detect the anomalies and draw them in plotly graph. A sample dataset is also uploaded in the Data folder.
## List of files:
###### main_file: Ths is a main file of this program.
###### SARIMAX_forecasting: Run the SARIMAX forecasting model.
###### anomalies_detect: Detect the anomalies from prediction and target data.
###### plot_graph: Plot the anomalies
